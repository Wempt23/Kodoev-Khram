HTML Tags Reference

<!DOCTYPE>
Declares the document type as HTML5.
<html>
Root element that wraps all HTML content.
<head>
Contains metadata, title, and links to stylesheets.
<body>
Contains visible page content.
<h1> to <h6>
Heading tags from largest to smallest.
<p>
Paragraph of text.
<a>
Hyperlink to other pages or resources.
<img>
Embeds an image.
<div>
Generic container for grouping content.
<span>
Inline container for text styling.
<ul> / <ol> / <li>
Unordered list, ordered list, and list items.
<table>
Creates a data table.
<form>
Container for form elements.
<input>
Input field for user data entry.
<button>
Clickable button element.
